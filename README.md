Useing the LiveCoinWatch API to store data in a MongoDB database. If it doesn't work, add an API key from LiveCoinWatch. Also, ensure the MongoDB URL is correctly set in the environment file.

Frontend: 
  - To run the frontend 
      1. cd frontend
      2. npm install
      3. npm run dev

Backend: 
  - To run the backend
    1. cd backend
    2. npm install 
    3. npm start
